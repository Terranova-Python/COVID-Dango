# COVID-Dango
Covid Stats represented with a Web-based application using the Django framework and charts.js
Created by Anthony Terrano and Marissa Vitols 2021

# IN THIS ORDER
# Open command prompt in the COVID-Django directory

1. Apply the migrations:
python manage.py migrate

2. Load the Data:
python manage.py loaddata countries.json cities.json

3. Run Server:
python manage.py runserver
